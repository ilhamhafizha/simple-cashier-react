# Simple Cashier React

## Fitur

- **Daftar Produk:** Menampilkan produk beserta harga.
- **Keranjang Belanja:** Menambah dan menghapus produk.
- **Struk Belanja:** Menampilkan ringkasan belanja.
- **Reset:** Mengosongkan keranjang belanja.
